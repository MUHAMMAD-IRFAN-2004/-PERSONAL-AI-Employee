## 🔧 Configuration Reference

### WhatsApp Settings
```env
WHATSAPP_HEADLESS=false              # Show browser window
WHATSAPP_CHECK_INTERVAL=30           # Check every 30 seconds
WHATSAPP_PRIORITY_SENDERS=Boss,Client  # Priority contacts
```

### Approval Settings
```env
AUTO_APPROVE_LOW_RISK=true           # Auto-approve low risk
HIGH_RISK_THRESHOLD=50               # High risk score threshold
```

## 🆘 Troubleshooting

### Common Issues

**Playwright not found:**
```bash
pip install playwright
playwright install chromium
```

**WhatsApp QR keeps expiring:**
- Set `WHATSAPP_HEADLESS=false`
- Delete `.whatsapp_browser_data/`
- Restart and scan faster

**Social media login fails:**
- Verify credentials in `.env`
- Set `SOCIAL_HEADLESS=false` to see browser
- May need 2FA/security challenge first login

**Approval system not routing:**
- Check if `approval_system.py` is running
- Verify directory permissions
- Check logs in `Logs/`

**Run status check:**
```bash
check_silver_status.bat
```
